LAPORAN PRAKTIKUM

Halaman ini berisi hasil praktikum pemrograman web menggunakan HTML, CSS, dan JavaScript.

Soal 1

CSS (Cascading Style Sheets) dan JavaScript digunakan karena keduanya memiliki peran penting dalam membangun tampilan serta perilaku situs web modern. CSS berfungsi untuk mengatur tata letak, warna, font, ukuran, serta gaya visual lainnya agar halaman web terlihat menarik, konsisten, dan mudah dibaca di berbagai perangkat. JavaScript digunakan untuk memberikan interaktivitas dan dinamika pada halaman, seperti menampilkan pop-up, mengganti tema, validasi formulir, serta merespons aksi pengguna tanpa perlu memuat ulang halaman. Dengan memadukan CSS dan JavaScript, pengembang dapat menciptakan halaman web yang tidak hanya indah secara tampilan, tetapi juga interaktif dan responsif terhadap pengguna.

Soal 2

Responsive Web Design (RWD) adalah pendekatan desain web yang bertujuan agar tampilan situs dapat menyesuaikan diri secara otomatis terhadap berbagai ukuran layar dan perangkat, seperti desktop, tablet, maupun smartphone. Tujuan utama RWD adalah memberikan pengalaman pengguna yang optimal, di mana konten tetap mudah dibaca dan navigasi tetap nyaman tanpa perlu menggulir atau memperbesar secara manual. Konsep ini biasanya diterapkan menggunakan media query, flexbox, atau grid layout pada CSS agar tata letak bisa berubah secara proporsional sesuai lebar layar.

Soal 3

Ketiga elemen ini bekerja secara saling melengkapi untuk menciptakan pengalaman web yang menarik dan fungsional: CSS bertanggung jawab terhadap tampilan visual, mengatur layout, warna, dan transisi agar halaman terlihat rapi dan nyaman. JavaScript menambahkan elemen interaktif, seperti tombol yang dapat mengubah mode tampilan, animasi, atau menampilkan/menyembunyikan elemen tertentu. RWD (Responsive Web Design) memastikan bahwa seluruh tampilan dan interaksi tersebut dapat beradaptasi dengan baik di berbagai ukuran layar. Kombinasi ketiganya memungkinkan halaman web menjadi dinamis, menarik, dan fleksibel — mampu memberikan pengalaman yang konsisten dan responsif bagi pengguna di semua perangkat.
